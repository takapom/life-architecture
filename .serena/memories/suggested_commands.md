# Suggested commands
- `rg --files` : list tracked files quickly.
- `rg -n "<pattern>" docs README.md CLAUDE.md` : search spec and architecture docs.
- `nl -ba <file>` : inspect docs with stable line numbers for review.
- `git status` : inspect working tree.
- Note: the repository currently has no package manifest, app scaffold, or test/lint scripts yet, so there are no project-specific run/build/test commands at this stage.
